# GuetHub App 一个使用 Flutter 开发的全平台 GUET 工具箱（课程表）

此项目为桂林电子科技大学课程表App项目，可以查看课程信息，成绩单，考试安排，评教等功能，此外还有论坛功能，可以发帖互动。
本人负责90%以上的代码开发，包括后端。
本项目App端代码已有10万行，后端4000行
本项目App端使用 Flutter 跨平台开发。
后端使用 Golang语言+mariadb+redis+minio对象存储。
本项目在学校有数千用户，包括安卓和iOS端

支持 Android IOS

下载地址 [https://www.guethub.net](https://www.guethub.net)
